종류 -> 0      # 0~6 테트리스 조각

	사각형(5,5,999999)
	사각형(5,6,999999)
	사각형(5,7,999999)
	사각형(5,8,999999)
종류=1 이면
	사각형(5,5,990000)
종류=2 이면
	사각형(5,5,9900)
종류=3 이면
	사각형(5,5,99)
종류=4 이면
	사각형(5,5,99)
종류=5 이면
	사각형(5,5,99)
종류=6 이면
	사각형(5,5,99)

###############################
사각형(세로, 가로, 색) 정의
	세로횟수 -> 0
	세로횟수 < 4 동안
		가로횟수 -> 0
		가로횟수 < 4 동안
			메모리[60000 + (세로*500) + (가로*5) + (세로횟수*100) + 가로횟수] -> 색
			가로횟수 -> 가로횟수 + 1
		세로횟수 -> 세로횟수 + 1
